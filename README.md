# my-skill
A list of skills

## Skills

### Weather Skill (`skill.py`)
An interactive skill that gives you a personalized weather-themed greeting based on your mood.

### Product Manager Skill (`product_manager_skill.py`)
An interactive skill that helps with common Product Manager tasks:
1. **Write a User Story** - Generates a user story in standard "As a / I want / So that" format with acceptance criteria.
2. **Prioritize Features (MoSCoW)** - Categorize features into Must / Should / Could / Won't have.
3. **Generate a Mini PRD** - Create a lightweight Product Requirements Document.
4. **Elevator Pitch Generator** - Build a structured elevator pitch using Geoffrey Moore's template.
